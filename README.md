# GazerV2
=========

Gazer Bot is a Discord made for only one Discord Server. It consist of many packages.

[Discord Server](https://discord.gg/5dGwF2T)

Packages
---------
- discord.js ( Main Libray )
- sqlite ( Main Database )
- moment ( Timer )
- weather.js ( Searching Weather )
